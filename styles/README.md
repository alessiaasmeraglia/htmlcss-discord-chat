# htmlcss-discord-chat

Replica in **HTML e CSS** del layout **mobile** della chat di Discord.

## Preview del progetto

Il progetto riproduce una schermata ispirata alla sezione messaggi di Discord in versione mobile, con:

- sidebar dei server
- header principale
- lista delle chat
- footer di navigazione
- effetti hover
- stato utente sugli avatar
- bottone rapido per nuova chat

## Obiettivo

L'obiettivo dell'esercizio è allenarsi con:

- struttura HTML
- layout con `flex`
- allineamenti
- spaziature
- gestione di sezioni verticali e orizzontali
- hover effects
- scrollbar in una sezione specifica
- utilizzo di `position: relative` e `position: absolute`

## Tecnologie usate

- HTML5
- CSS3
- Git
- GitHub
- Visual Studio Code
